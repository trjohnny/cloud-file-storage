# MinIO Cloud Deployments for Cloud Computing Course

This repository hosts the practical exercises for the Cloud Computing course for the academic year 2023/2024. It is structured to guide through the deployment of a cloud-based file storage system using MinIO, covering both a docker-compose and a k8s setup.

## Overview

- **CloudBasic**: This directory focuses on a Docker Compose setup for MinIO.

- **CloudAdvanced**: This directory focuses on a Kubernetes setup for MinIO.

## Instructions

Each directory contains a `README.md` with detailed instructions on deploying MinIO within the respective environment.
